# CityWide HRM Demo

A comprehensive Human Resource Management system demo showcasing key features for GitHub display.

## Features

### 🏢 Dashboard
- Overview of key metrics
- Employee statistics
- Shift completion rates
- Visual progress indicators

### 👥 Employee Management
- Add, edit, and delete employees
- Employee status tracking
- Department and position management
- Comprehensive employee database

### 📅 Scheduling
- Shift scheduling and management
- Employee shift assignments
- Location-based scheduling
- Time tracking

### 📊 Analytics
- Revenue trend analysis
- Employee growth metrics
- Performance dashboards
- Interactive charts and graphs

### 🎯 Applicant Tracking System
- Job posting management
- Candidate pipeline tracking
- Recruitment workflow
- Application status management

## Technology Stack

- **Frontend**: React 18 with TypeScript
- **UI Library**: Ant Design
- **State Management**: Redux Toolkit
- **Charts**: Recharts
- **Build Tool**: Create React App

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/hrm-demo.git
   cd hrm-demo
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`
